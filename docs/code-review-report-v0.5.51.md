# Code Review — v0.5.51

*最后更新: 2026-03-25 | 范围: 入口卡片强调收敛 / 入口落点可见性 / E2E 稳定性*

## 范围

- 入口节点卡片样式：强调从“大面积强调条”收敛为“顶部细线 + 小色点”（灰阶为主）。
- 入口节点默认落点：视口锚定坐标补齐 `vw/vh`，在 `offset=0` 的全新环境也能在视野内出现。
- Playwright E2E：注入 `window.__E2E__`，在 E2E 环境关闭物理模拟与漂浮动画，避免节点持续微动导致点击不稳定。

## 结论

| 项 | 结果 |
|----|------|
| 线上交互与视觉 | 保持：入口卡片仅做更克制的强调，不改变信息结构与操作方式 |
| 入口可见性 | 提升：全新环境/测试环境可稳定落在视野内，避免被顶出视口 |
| E2E 稳定性 | 提升：关闭测试环境动画/物理后，点击与 hover 不再受“微动”影响 |
