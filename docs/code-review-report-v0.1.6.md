## 关键实现说明

### 1. 岛屿中心布局算法 ✅

**文件**: `canvasStore.ts` 中的 `addNode`。
**逻辑**: 
- 计算所属 Category 的所有节点质心 (Centroid)。
- 以质心为起点，进行轨道碰撞检测螺旋搜索。
- 确保节点间距的同时，维持板块的紧凑性。

### 2. API 安全管理 ✅

**文件**: `SettingsModal.tsx` 与主进程 `index.ts`。
**安全**:
- 使用 Electron 的 `safeStorage` 加密存储 API Key。
- 其他非敏感配置（Base URL, Model）存储于 `settings.json`。

### 3. 全局 Framer Motion 动效 ✅

**覆盖组件**:
- `AnswerModal.tsx`: 全屏弹窗缩放与淡入。
- `ConversationSidebar.tsx`: 抽屉式弹簧滑出。
- `SettingsModal.tsx`: 物理反馈弹窗。
- `NodeCard.tsx`: 呼吸灯与布局动画。
